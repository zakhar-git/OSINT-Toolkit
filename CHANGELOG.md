## Unreleased

### Added

- Curation policy with explicit selection, rejection, maintenance-review, service, and TODO rules.
- Tool submission and pull request gates for maintenance evidence and professional usage signals.

### Changed

- Replaced weaker or stale current recommendations with verified, actively maintained or officially documented alternatives: TDLib, GHunt, Have I Been Pwned API, IPinfo CLI, 4CAT, Zeeschuimer, Zotero, whisper.cpp, OpenCTI, and MISP.
- Updated validation to compare README statistics with actual generated card counts instead of treating catalog size as a quality target.

### Removed

- Removed h8mail, Holehe, Telethon, ZMap, Social Analyzer, snscrape, SpiderFoot, OpenAI Whisper reference repository, Recon-ng, and OSINT Framework from current curated recommendations after applying the stricter maintenance and professional-use criteria.

## 1.0.0 - 2026-07-07

### Added

- Nineteen category guides with thirty-eight standardized tool cards.
- Nine investigation workflows with evidence and stop conditions.
- Contribution, security, conduct, architecture, style, and review documentation.
- Structured issue forms and a pull request template.
- Markdown lint, broken-link, TOC, and repository-consistency automation.
